# Document Extraction Service

This service will parse W-2 documents and transform raw intake files into structured tax data.

## Responsibilities

- Extract fields from W-2 images and PDFs
- Normalize payroll and employer data
- Emit structured extraction records for validation

## Layout

- `service.py` - placeholder service entrypoint
- `requirements.txt` - Python dependencies
- `README.md` - service description
