# Tax Mapping Service

This service maps validated tax data into 1040 payloads and tax intelligence artifacts.

## Responsibilities

- Transform extracted W-2 data for 1040 preparation
- Build tax input models for downstream analysis
- Support multiple tax scenarios and state/local mapping

## Layout

- `service.py` - placeholder service entrypoint
- `requirements.txt` - Python dependencies
- `README.md` - service description
