# Data Validation Service

This service evaluates extracted tax data for correctness, completeness, and compliance.

## Responsibilities

- Validate W-2 fields and numeric values
- Apply business rules and tax compliance checks
- Route invalid records to review workflows

## Layout

- `service.py` - placeholder service entrypoint
- `requirements.txt` - Python dependencies
- `README.md` - service description

## Deployment

- Infrastructure: `scripts/services/data-validation/deploy.ps1`
